说明

CAS NET 是中科院内部 IP 控制网关登录客户端，支持 Linux 和 Windows 双系统。此软件完全使用 Python 语言写成，同时拥有命令行和图形界面，使用简单，安装方便，实乃中国科学院 IP 网关用户居家旅行必备之良品 :)。

关于 eYouIPB 的分析，以及该软件和 eYouIPB 的比较，请看这里。

软件特性

  1. 同时支持 Linux 和 Windows 操作系统！
  2. 提供各种格式的安装包，方便安装过程。
  3. 客户端同时具有命令行和图形界面，满足不同用户需要。
  4. 可设置选项多，可保存用户设置，登录简单快捷。
  5. 纯 Python 编程，修改简单，扩展性强。
  6. 开放源代码，确保程序无后门。

最新版本 1.3-1(2009年2月11日发布) 更新

  1. 增加了单击更换登录模式功能。
  2. 增加了自动断线重连功能。
  3. 增加了余额不足提醒功能。
  4. 解决了以前版本的一些 BUG。

对某些 Windows 用户的劝告

如果您已经习惯而且很喜欢使用 eYouIPB，您实在没必要来尝试这个软件。您也可以先读读 eYouIPB 和 CASNET 的差别再做决定。

如何选择合适自己的安装包

看过了（懒得看），我都懂！请直接到 Google Code 项目页下载。

if 您是 Linux 用户并且拥有 Python 和 PyGtk 支持(一般的 Linux 发行版都有):
  if 您使用 RedHat系列的 Linux 系统(Fedora, RHEL, CentOS,...):
    请下载 casnet-1.3-1-all.rpm
  elif 您使用 Debian 系列的 Linux 系统(Debian, Ubuntu,...):
    请下载 casnet-1.3-1_all.deb
  else:
    请下载 casnet-1.3-1_src.tar.gz(源代码包)
elif 您是 Windows 用户:
  if 您是 Python 开发者并确信您 Windows 系统里已经安装 Python 和 PyGtk:
    请下载 casnet-1.3-1_win_PyGtk_Installed.zip
  else:
    请下载 casnet-1.3-1_win.exe
